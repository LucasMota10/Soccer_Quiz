# Soccer Quiz 

Este é um projeto Flutter desenvolvido aplicando os princípios S.O.L.I.D.
O projeto atualmente conta com sistema de Login, Cadastro, Recuperação de Senha, Convite de Usuários e Banco de Dados Local (SQLite).

 ## Pré-requisitos

- Flutter SDK
- Android Studio
- vs code

## Como executar

### 1. Instalar dependências

Abra o terminal na pasta raiz do projeto e execute o comando abaixo para baixar as bibliotecas (sqflite, path, etc):

flutter pub get

### 2. Rodar o aplicativo

- Abra a pasta do projeto
- Selecione o emulador em Flutter: Select Device
- Execute o comando: flutter run no terminal
